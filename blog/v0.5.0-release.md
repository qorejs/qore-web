# Qore v0.5.0 Release

Qore v0.5.0 focused on the foundation for server rendering, streaming output, and a smaller runtime surface.

## Highlights

- Better runtime structure for streaming response
- Smaller client bundle
- Early SSR and hydration groundwork
- Clearer API boundaries around `stream()` and signals

## Upgrade

```bash
pnpm add @qorejs/qore@latest
```

## Next

The next phase is deeper provider support, stronger type tests, and more browser-level regression coverage for the official demos.
