🎉 Qore v0.5.0 正式发布！ ​

我们很高兴地宣布 Qore v0.5.0 正式发布！这个版本带来了完整的 SSR（服务端渲染）支持、性能优化和多个新特性。

🚀 新特性 ​

完整的 SSR 支持 ​

v0.5.0 引入了完整的 SSR 能力，支持：

  • ✅ 流式 SSR
  • ✅ Suspense 边界
  • ✅ 数据预取
  • ✅ 混合渲染（Hydration）
typescript
import { renderToStream } from '@qore/ssr';

const stream = await renderToStream(App, {
  prefetch: true,
  suspense: true
});

性能优化 ​

  • 渲染速度提升 30%
  • 内存占用降低 25%
  • 包体积优化至 5KB (gzipped)

新增 API ​

  • lazy() - 懒加载组件
  • createErrorBoundary() - 错误边界
  • retry() - 自动重试机制
