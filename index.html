Live primitive
stream = signal

AI output is reactive state from the first byte.

Token-level UI

Only the dependent text node updates while tokens arrive.

Provider-ready

OpenAI, Anthropic, and generic SSE adapters fit one primitive.

The point

不是把 token 搬进状态，而是让 token 自己成为状态。

Qore 的核心路径只有两步：声明 stream，然后让 UI 订阅 signal。

import { h, stream, text } from '@qorejs/qore'

const response = stream(openAI.chat(prompt))

export function App() {
  return h('main', {},
    text(() => response())
  )
}
