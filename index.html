Streaming Response Framework

Qore

让数据像水流一样自然地流进 UI。

stream 是数据流动的方式，signal 是 UI 响应变化的方式。Qore 把它们变成同一个 primitive。

快速开始看流式响应GitHub
Live primitive
stream = signal

AI output is reactive state from the first byte.

Token-level UI

Only the dependent text node updates while tokens arrive.

Provider-ready

OpenAI, Anthropic, and generic SSE adapters fit one primitive.

The point

不是把 token 搬进状态，而是让 token 自己成为状态。

一个 AI 聊天页最核心的代码，只需要声明数据流和视图依赖。

Qore
import { h, stream, text } from '@qorejs/qore'

const answer = stream(openAI.chat('hello'))

export function App() {
  return h('main', {},
    text(() => answer())
  )
}
